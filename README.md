## 📌 Overview

This project is a personal cybersecurity portfolio designed to showcase:
- Ethical hacking skills
- Cybersecurity tools and technologies
- Real-world security projects
- Security credentials and certifications (including Hack The Box)
- Field experience (internships, hackathons, and activities)

The portfolio follows a red-team inspired, terminal-based design to reflect a hacker-centric mindset while maintaining a professional presentation.


## ✨ Features

### 🎨 Design & UI
- Terminal-style interface with window controls
- Dark red cyber security theme with glow effects
- Fully responsive layout for all devices
- Animated hacker-style background

### 🔧 Interactive Elements
- Animated skill progress bars on scroll
- Tool cards with hover tooltips

- Terminal typing animation in contact section
- Certificate trophy cards with hover glow

## ⚙️ Customization

### Personal Information
- Update name and role in the hero section
- Modify contact details and social links
- Edit certification details

### Skills
- Change skill levels using the `data-level` attribute
- Update skill names directly in HTML

### Projects
- Modify project titles and descriptions
- Update technology tags and GitHub links

### Certificates
- Replace certificate images in `assets/certificates/`
- Ensure filenames match those used in HTML

## 🎮 Usage Guide

### Navigation
- Use top navigation buttons to switch sections
- Keyboard shortcuts:
  - Ctrl + 1 → Identity
  - Ctrl + 2 → Capabilities
  - Ctrl + 3 → Operations
  - Ctrl + 4 → Credentials
  - Ctrl + 5 → Certifications
  - Ctrl + 6 → Trophies
  - Ctrl + 7 → Experience
  - Ctrl + 8 → Contact
- Press ESC to reset skill animations

### Terminal Controls
- Red button → Termination screen
- Yellow button → Minimize window
- Green button → Fullscreen mode

### Interactive Behavior
- Skill bars animate on scroll
- Tooltips appear on hover
- Terminal typing effect activates in contact section

## 🐛 Troubleshooting

### Common Issues
- Images not loading → Check file paths and filenames
- Animations not working → Verify JavaScript is enabled
- Layout issues → Test viewport settings and screen sizes

### Debugging
- Open Developer Tools (F12)
- Check Console for errors
- Inspect elements for CSS issues
- Monitor Network tab for missing files


## 🔄 Maintenance & Updates

### Update Skills
- Modify `data-level` values in skill elements

### Add New Projects
- Duplicate an existing project card
- Update project details and links

### Add Certifications
- Add a new certificate image to `assets/certificates/`
- Create a corresponding certification item and trophy card in `Portfolio.html`

### Update Experience
- Add new internships, hackathons, or activities under the `#experience` section in `Portfolio.html`
- Use the structured CSS classes (`internship`, `hackathon`, `activity`) to maintain visual consistency
